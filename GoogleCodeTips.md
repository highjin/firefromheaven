# Wiki #
  * Wiki是个博大精深的东西，其实学会以后很简单。可以看看[Google的Wiki帮助](http://code.google.com/p/support/wiki/WikiSyntax)
  * 要人工换行的话，需要打两次回车
  * 显示列表（就像现在这个）在每行开头打“` * `”（注意`*`两边的半角空格）
  * 显示目录：
```
<wiki:toc max_depth="1" />
```
